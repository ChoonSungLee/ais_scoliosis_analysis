# AIS Scoliosis Analysis (Complete Pooling Model)

이 프로젝트는 2,700례의 AIS 데이터를 활용하여 척추경 넓이를 분석하는 **완전 풀링 모델** 연구입니다. 모든 환자의 데이터를 하나의 큰 그룹으로 간주하여 기초적인 통계적 특성을 파악하는 것을 목표로 합니다.

## ✅ 풀링 모델 완벽 습득을 위한 핵심 체크리스트

### 1. 가능도(Likelihood)의 이해
- [ ] 2,700례 전체 데이터가 단일 정규분포 `normal(mu, sigma)`를 따른다고 가정했을 때, MCMC가 안정적으로 수렴하는지 확인.
- [ ] Rhat 값이 1.1 미만인지, Effective Sample Size가 충분한지 진단.

### 2. 사전분포 민감도 분석
- [ ] `mu ~ normal(5, 3)` 설정이 사후분포에 미치는 영향 파악.
- [ ] 사전분포의 표준편차를 변경했을 때 결과의 견고성(Robustness) 확인.

### 3. 잔차(Residual) 분석 및 변동성 확인
- [ ] 모델에서 설명되지 않는 변동성(`sigma`)의 크기 관찰.
- [ ] 이 전체 변동성이 나중에 계층 모델에서 '환자 간 편차'로 분리될 것임을 인지.

### 4. 모델의 한계 및 그룹별 편차 관찰 (추가 항목)
- [ ] **데이터 시각화**: 전체 평균(`mu`)이 특정 환자군이나 특정 척추 레벨의 실제 데이터를 얼마나 잘 대표하지 못하는지(과소/과대 추정) 확인.
- [ ] **개별성 무시의 비용**: 모든 모수가 같다고 가정했을 때 발생하는 오차의 패턴을 분석하여 '계층 모델의 필요성'을 스스로 입증해 보기.

## 📌 향후 계획
